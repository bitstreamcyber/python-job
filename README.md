# Desktop Support Toolkit

A collection of Python scripts and utilities for IT Support and Desktop Administration.

## Features

* System Information Collection
* Windows Cleanup & Maintenance
* Network Troubleshooting
* Software Inventory
* BitLocker Status Checks
* Printer Troubleshooting

## Requirements

* Windows 10/11
* Python 3.x
* Administrator privileges (for some scripts)

## Usage

```bash
python script_name.py
```

Example:

```bash
python windows_cleanup.py
```

## Purpose

This repository demonstrates practical IT automation skills using Python to streamline common support and troubleshooting tasks.
